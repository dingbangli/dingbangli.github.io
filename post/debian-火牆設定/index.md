# Debian - 火牆設定


<!--more-->
"紀錄一下 因火牆設定 導致無法建立 ssh 連線"

{{< admonition >}}
# 安裝 UFW

    apt install ufw
{{< /admonition >}}

{{< admonition >}}
# 關閉 UFW

    ufw disable
{{< /admonition >}}

{{< admonition >}}    
# 開啟 UFW

    ufw enable
{{< /admonition >}}
