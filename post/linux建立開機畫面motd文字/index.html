建立開機畫面 motd 文字

使用 vim 替換 ^[ 為 轉義 字符

在 vim 內按 :

輸入以下內容

%s/\^\[/

(不要點擊Enter)

輸入 Ctrl+v 然後點擊 Esc 轉換字符

最後輸入 /g ，點擊 Enter

如果操作正確 ^[ 則從灰色變為藍色，如果沒有點擊 u 返回

另一種方法 ( 使用 echo )

EX:
echo $'\e[1;37m' > myfile

各符號代表意思 :

符號 意思
%s 搜索
/ 第一項
^[ 轉義版本 ^[
/ 第二項，使用鍵盤插入實際 Esc
/g 重複所有出現
