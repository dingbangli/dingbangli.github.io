# CentOS - 目錄沒顯示顏色


<!--more-->
"紀錄一下如何更改 Linux 系統目錄顏色"

# 語法:

```sql

yum install coreutils --allowerasing && or dnf install
    
scp DIR_COLORS root@172.17.0.2:~
    
mv DIR_COLORS ~/.dir_colors
    
reboot

```
