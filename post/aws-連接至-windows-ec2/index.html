“如何使用RDP協議遠端桌面連線至 EC2-Windows”

Note

在新建立的 Instances 右鍵選擇 Connect

Note

選擇 RDP client 並點選 Get password

Note

選擇 Browse (瀏覽) 並選取製作 instance 的金鑰 (.pem) 並選擇 Decrypt Password (解密密碼)

Note

使用後台給的亂數密碼建立遠端桌面連線 (帳號預設都是 Administrator)
