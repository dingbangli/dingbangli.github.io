# Linux - Nodejs 安裝紀錄


***
***

```sql
yum install epel-release

yum install nodejs

wget https://nodejs.org/download/release/v8.9.4/node-v8.9.4-linux-x64.tar.gz

tar --strip-components 1 -xzvf node-v* -C /usr/local

node -v

npm -v
```
